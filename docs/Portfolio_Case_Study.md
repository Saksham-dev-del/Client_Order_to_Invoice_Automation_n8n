# Portfolio Case Study

## Title

Client Order-to-Invoice Automation using n8n

## Objective

Build an automation pipeline that converts new e-commerce orders into invoices and confirmations without manual work.

## Tools

- n8n
- Airtable
- Twilio WhatsApp
- Webhooks
- PDF API

## Interview Pitch

“I built an n8n automation workflow where a webhook receives a new order, validates the order payload, calculates invoice totals, generates invoice HTML, stores the order in Airtable, and sends a WhatsApp confirmation through Twilio. This removes repetitive manual operations and can save around 12+ hours per week for a small e-commerce client.”
